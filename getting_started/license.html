LICENSE

If you want others to eventually work on your content (or not) you need a license file. A license file lets others know what is and isn't permitted with your code. This is a good way to protect your content and ensure that the appropriate credit is given. Check out ChooseALicense.com for recommendations based on requirements..

Add your license file to GitHub

Like the README, you can update your LICENSE locally, or directly on GITHUB. To edit your LICENSE on GitHub, Create a new file and edit it. Then select your license from the dropdown.

Working from your Local Machine

We get it...

Working on everything through github is not the best way to get things done.

No one expects you to only work from the website (we actually recommend that you don't).

The best approach to working with your code is to clone your code to your local machine where you can use the tools and technology that you are comforatble with.

On the Repository Root Page select the Clone or Download button.

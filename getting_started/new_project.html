Creating Your New Project

At the top of your dashboard, select the "+" icon, then select New Repository.

Enter a name for you Repository and a description for your project. This will help you remember you long-sitting projects.

Public vs Private Repositories

Public Repositories (more common) are visible to the public. No need to be afraid. People cannot make changes to your code by-default unless they are a contributor. They will be able to view your code and any changes. Private Repositories are only visible to the people that you allow to view it.

NOTE: If privacy is an important feature, I would suggest that you consider using a more privacy focused Source Control Platform, like BitBucket, or host your own git server.
